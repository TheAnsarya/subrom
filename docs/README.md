# Subrom Documentation

Welcome to the Subrom documentation. Subrom is a ROM management and verification toolkit for retro gaming enthusiasts.

## What is Subrom?

Subrom helps you:
- **Verify ROMs** against DAT files from No-Intro, TOSEC, MAME, and more
- **Organize** your ROM collection with intelligent renaming and folder structures
- **Track** ROMs across multiple drives, including offline storage
- **Scan** archives (ZIP, 7z, RAR) without extraction
- **Identify** duplicates, bad dumps, and missing ROMs

## Quick Start

1. **Install Subrom** - Download and run the installer
2. **Add DAT Files** - Import DAT files from your preferred provider
3. **Scan Your ROMs** - Point Subrom at your ROM folders
4. **Verify** - See which ROMs match, are missing, or are duplicates
